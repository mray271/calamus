# Code of Conduct

Calamus follows the [GNOME Code of Conduct](https://conduct.gnome.org/).

## Scope

This applies to all Calamus project spaces, including:
- GitHub issues, pull requests, and discussions
- release and support communication
- any other project-managed public community channels

## Reporting

Report Code of Conduct incidents to the GNOME Code of Conduct committee using
the reporting instructions at <https://conduct.gnome.org/reporting/>.

For project-specific follow-up, you may also contact the Calamus maintainer by
opening a private GitHub security advisory draft or by requesting maintainer
contact through a GitHub issue.
